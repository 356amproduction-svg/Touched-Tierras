## 🌐 How to Use

1. **View locally** — Open `index.html` in any web browser (Chrome, Safari, Firefox)
2. **Upload to web hosting** — Upload the entire `TouchedTierras_Website` folder to your hosting provider
3. **Adding new gallery photos** — Drop new `.jpg` or `.png` files into the `images/gallery/` folder and update `index.html` to reference them
